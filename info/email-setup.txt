FUTURE PLAN: MIGRATE TO NAMECHEAP EMAIL
===============================================================================

REASON: Already paid for Namecheap Private Email subscription

MIGRATION STEPS NEEDED:
1. Create mailbox in Namecheap Private Email dashboard
   - Create info@creativekidsmusic.org mailbox
   - Create jack@creativekidsmusic.org mailbox (if needed)

2. UPDATE DNS RECORDS IN VERCEL:
   - Log into Vercel Dashboard
   - Go to Project > Domains > DNS Records
   - REMOVE current Zoho records:
     * Delete MX record: mx3.zoho.com
     * Delete TXT record: v=spf1 include:zohomail.com ~all
     * Delete DKIM record: zmail._domainkey

   - ADD Namecheap records:
     * Add MX: mx1.privateemail.com (Priority: 10)
     * Add MX: mx2.privateemail.com (Priority: 10)
     * Add TXT: v=spf1 include:spf.privateemail.com ~all

3. Wait for DNS propagation (up to 4 hours, usually faster)

4. Test new email by sending to info@creativekidsmusic.org

5. Optional: Export/backup any important emails from Zoho before canceling

IMPORTANT NOTES:
- Changing DNS records will BREAK existing Zoho email
- Do this during low-traffic time
- Test thoroughly before canceling Zoho
- Can configure Namecheap email in phone/desktop apps via IMAP/SMTP
