- After successful auth, "Welcome back" box collapses or shows checkmark
- Submit button creates new registration linked to existing account

## Notes

### Inline Login UX
- Email check happens on blur (when field loses focus)
- Use a subtle loading indicator while checking
- Transition should be smooth (fade/slide, not jarring)
- Form fields remain editable during the entire process
- After successful login, collapse auth section to single line confirmation

### Mobile Considerations
- Modals should be full-screen on mobile
- Cards stack vertically
- Touch-friendly button sizes (44px minimum)

### Error States (not shown)
- Invalid password: red border + message below field
- Network error: toast notification with retry
- Session expired: redirect to login with message
